## Installation

No installation required!
1.  Download `PDFSimpleToolset.exe`.
2.  Double-click to run.

## Usage Guide

### Merging Files
1.  Click **Merge PDFs** on the Home screen.
2.  Drag and drop your files into the staging area.
3.  Click **Merge Files** and choose where to save the result.

### Splitting a Document
1.  Click **Split PDF**.
2.  Select your file.
3.  Enter a page range (e.g., `1-3, 5`) or leave empty to split every page.
4.  Click **Split PDF**.

### Themes
The application runs in a **Premium Dark Mode** optimized for focus and reduced eye strain.
